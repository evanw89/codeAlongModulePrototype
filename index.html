What is HTML?

HTML stands for Hypertext Markup Language.

I know, it's a mouthful, but we'll break that down in a minute. First, the big picture: HTML is the structure behind any web page you've ever visited. HTML is the skeleton, or the scaffolding.

The first step towards creating websites should be to learn HTML, but don't worry; there's actually a very finite amount of HTML you'll ever need to learn! To begin, we're going to walk you step-by-step through a series of coding scenarios like you see below. You can write your own code in the document on the left and see it rendered immediately in the window on the right!

The way HTML works is, websites are put together using HTML elements. If you look in the code document below, you'll see an empty HTML h1 element. I say empty because elements like this are normally supposed to contain some content, such as text.

Go ahead and write your name in between the two h1 tags (between <h1> and </h1>) and see what happens!
